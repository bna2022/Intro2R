4.1 Kontrollstrukturen

Mit Kontrollstrukturen kann man:

  • mehrere Befehle zusammenfassen,
  • Entscheidungen treffen,
  • Abläufe wiederholen.

4.1.1 Anweisungsblöcke

Ein Anweisungsblock fasst mehrere Befehle zusammen. In R werden Blöcke mit geschweiften Klammern { } definiert.

4.1.1.1 Syntax

{
  Befehl1
  Befehl2
  Befehl3
}

Beispiel: Berechnung eines BMI

{
  gewicht <- 72
  groesse <- 1.78
  bmi <- gewicht / groesse^2
  bmi
}
[1] 22.7244

Alle Befehle werden nacheinander ausgeführt. Der letzte Ausdruck wird ausgegeben.

4.1.2 Bedingte Anweisungen und Verzweigungen

Mit Bedingungen kann Code nur dann ausgeführt werden, wenn bestimmte Kriterien erfüllt sind.

4.1.2.1 if-Anweisung

Syntax

if (Bedingung) {
  Befehle
}

Beispiel: Fieber erkennen

temperatur <- 38.5

if (temperatur >= 38) {
  print("Fieber vorhanden")
}
[1] "Fieber vorhanden"

4.1.2.2 if … else

Syntax

if (Bedingung) {
  Befehle_wenn_wahr
} else {
  Befehle_wenn_falsch
}

Beispiel: Blutdruckbewertung

systole <- 128

if (systole >= 140) {
  print("Hypertonie möglich")
} else {
  print("Normalbereich")
}
[1] "Normalbereich"

4.1.2.3 if … else if … else

Mit dieser Struktur können mehrere Fälle unterschieden werden.

Beispiel: BMI-Klassifikation

bmi <- 27.4

if (bmi < 18.5) {
  print("Untergewicht")
} else if (bmi < 25) {
  print("Normalgewicht")
} else if (bmi < 30) {
  print("Übergewicht")
} else {
  print("Adipositas")
}
[1] "Übergewicht"

4.1.2.4 Wichtige Vergleichsoperatoren

Operator Bedeutung
== gleich
!= ungleich
> größer
< kleiner
>= größer gleich
<= kleiner gleich

4.1.2.5 Logische Operatoren

Operator Bedeutung
& und
| oder
! nicht

Beispiel

alter <- 67
raucher <- TRUE

if (alter > 65 & raucher) {
  print("Erhöhtes Risiko")
}
[1] "Erhöhtes Risiko"

4.1.3 Schleifen

Schleifen wiederholen Befehle mehrfach.

4.1.3.1 for-Schleife

Die for-Schleife wird verwendet, wenn bekannt ist, wie oft etwas wiederholt werden soll.

Syntax

for (variable in werte) {
  Befehle
}

Beispiel: Messwerte ausgeben

werte <- c(4.2, 5.1, 6.0)

for (wert in werte) {
  print(wert)
}
[1] 4.2
[1] 5.1
[1] 6

Beispiel: Mittelwert mehrerer Proben berechnen

proben <- c(8.1, 7.9, 8.4, 8.0)

summe <- 0

for (wert in proben) {
  summe <- summe + wert
}

mittelwert <- summe / length(proben)
mittelwert
[1] 8.1

4.1.3.2 while-Schleife

Die while-Schleife läuft so lange, wie eine Bedingung wahr ist.

Syntax

while (Bedingung) {
  Befehle
}

Beispiel: Zellwachstum

zellen <- 100

while (zellen < 1000) {
  zellen <- zellen * 2
  print(zellen)
}
[1] 200
[1] 400
[1] 800
[1] 1600

4.1.3.3 repeat-Schleife

Die repeat-Schleife läuft unbegrenzt, bis sie mit break beendet wird.

Beispiel

wert <- 1

repeat {
  wert <- wert * 2
  print(wert)

  if (wert > 100) {
    break
  }
}
[1] 2
[1] 4
[1] 8
[1] 16
[1] 32
[1] 64
[1] 128

4.1.3.4 break und next

4.1.3.4.1 break

break beendet eine Schleife sofort.

for (i in 1:10) {
  if (i == 5) {
    break
  }
  print(i)
}
[1] 1
[1] 2
[1] 3
[1] 4
4.1.3.4.2 next

next überspringt den aktuellen Schleifendurchlauf.

for (i in 1:5) {
  if (i == 3) {
    next
  }
  print(i)
}
[1] 1
[1] 2
[1] 4
[1] 5

4.1.4 Typische Fehler

4.1.4.1 Fehlende Klammern

x <- 5
if (x > 5)
  print(x)

Besser:

x <- 6
if (x > 5) {
  print(x)
}
[1] 6

4.1.4.2 Endlosschleife

while (x < 10) {
  print(x)
}

Hier wird x nie verändert. Dadurch endet die Schleife nicht.

4.1.5 Zusammenfassung

Struktur Zweck
{} Mehrere Befehle zusammenfassen
if Bedingte Ausführung
if ... else Zwei Alternativen
if ... else if Mehrere Fälle
for Wiederholung mit bekannter Anzahl
while Wiederholung mit Bedingung
repeat Wiederholung bis break

5 apply-Funktionen

Die apply-Familie ersetzt oft Schleifen.

Vorteile:

  • kürzerer Code
  • übersichtlicher
  • oft schneller
  • leichter lesbar

5.1 apply()

Für Matrizen oder Data Frames.

Syntax

apply(objekt, margin, funktion)
  • margin = 1 → zeilenweise
  • margin = 2 → spaltenweise

Beispiel: Mittelwert pro Spalte

daten <- matrix(c(5, 6, 7, 8, 9, 10), nrow = 3)

apply(daten, 2, mean)
[1] 6 9

Beispiel: Maximum pro Zeile

apply(daten, 1, max)
[1]  8  9 10

5.2 lapply()

Gibt immer eine Liste zurück.

werte <- list(a = 1:3, b = 4:6)

lapply(werte, mean)
$a
[1] 2

$b
[1] 5

5.3 sapply()

Versucht das Ergebnis zu vereinfachen.

sapply(werte, mean)
a b 
2 5

5.4 tapply()

Wendet Funktionen auf Gruppen an.

Beispiel: Mittlere Glucose nach Gruppe

glucose <- c(90, 100, 130, 140)
gruppe <- c("Kontrolle", "Kontrolle", "Diabetes", "Diabetes")

tapply(glucose, gruppe, mean)
 Diabetes Kontrolle 
      135        95

5.5 mapply()

Mehrere Vektoren gleichzeitig.

mapply(function(x, y) x + y,
       c(1,2,3),
       c(10,20,30))

6 Funktionales Programmieren

Hier werden Funktionen wie Daten behandelt:

  • Funktionen an Funktionen übergeben
  • Funktionen erzeugen Funktionen
  • anonyme Funktionen verwenden

6.1 Anonyme Funktionen

sapply(1:5, function(x) x^2)
[1]  1  4  9 16 25

Beispiel: Log-Transformation

werte <- c(1, 10, 100)

sapply(werte, function(x) log10(x))
[1] 0 1 2

6.2 Funktionen als Argumente

analyse <- function(x, FUN) {
    FUN(x)
}

analyse(c(1, 2, 3), mean)
[1] 2
analyse(c(1, 2, 3), sd)
[1] 1

6.3 Funktionen erzeugen Funktionen

multiplier <- function(faktor) {
    function(x) x * faktor
}

mal2 <- multiplier(2)
mal5 <- multiplier(5)

mal2(10)
[1] 20
mal5(10)
[1] 50

6.4 Typische Fehler

6.4.1 Funktion nicht aufrufen

bmi

statt:

bmi(72, 1.78)

6.4.2 Falscher Rückgabetyp bei lapply

lapply(df, mean)

liefert Liste, kein Vektor.

Falls Vektor gewünscht:

sapply(df, mean)
